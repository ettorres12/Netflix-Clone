# Netflix-Clone

🚀 Projeto inspirado na Netflix (UI Clone)

Este projeto marca o início de uma aplicação que será destaque no meu portfólio. Com foco em aprendizado prático e construção de interfaces modernas, ele recria a tela de seleção de perfis inspirada na Netflix.

💡 Utilizando Inteligência Artificial como apoio, o desenvolvimento foi otimizado para ser mais rápido, eficiente e focado em boas práticas de front-end.

🛠️ Tecnologias utilizadas:

HTML
CSS
JavaScript

📚 Objetivo:
Aprimorar habilidades em desenvolvimento web, manipulação de DOM e criação de interfaces responsivas e interativas.



Link do projeto: https://ettorres12.github.io/Netflix-Clone/
